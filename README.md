# Chat historia de la computación y la IA

Proyecto de la materia Modelizado de Sistemas IA.

## Tecnologías
- Streamlit (interfaz)
- Ollama + Llama3 (modelo IA)
- JSON (base de conocimiento)

## Integrantes
- Arnaldo
- Eze
- Maxi

## Objetivo
Desarrollar un chatbot especializado en la historia de la computación y de la inteligencia artificial.
